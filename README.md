## 🛠️ Tools & Fitur Excel yang Digunakan

| Fitur | Untuk Apa |
|-------|-----------|
| **Power Query** | Merger beberapa data yang terpisah dengan kode unik |
| **PivotTables & PivotCharts** | Merangkum data besar jadi ringkasan yang bermakna |
| **Slicers** | Membuat filter interaktif di dashboard |
| **VLOOKUP / HLOOKUP** | Menggabungkan data dari tabel berbeda |
| **IF & Calculated Field** | Membuat kolom bantuan untuk analisis lebih dalam |
| **Charts (Bar, Line, Pie)** | Visualisasi tren dan perbandingan |
